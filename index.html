Obamify Image Transformation API

A Flask-based HTTP service that blends user supplied images with the classic Barack Obama portrait. The project is now fully implemented in Python.

Quick start

  1. Run the application locally:
    python wsgi.py
  2. Send a transformation request:
    curl -X POST http://localhost:8000/api/transform -H "Content-Type: application/json" -d '{"source_image": "<base64 png>"}'

The API returns either a base64 encoded PNG or GIF payload, or the raw binary bytes when response_format is set to "binary".

Endpoints

GET /health

Health probe returning {"status": "ok"}.

POST /api/transform

Perform the image transformation.

Field Type Description
source_image base64 PNG/JPEG Required. The source image to transform.
target_image base64 PNG/JPEG Optional custom target portrait. Defaults to assets/pfp_transparent.png.
blend_ratio float (0-1) Intensity of the target portrait. Defaults to 0.65.
max_dimension int Maximum width/height in pixels. Defaults to 1024.
make_gif bool Create an animated GIF instead of a PNG.
gif_frame_count int Frames when generating a GIF. Defaults to 12.
gif_duration int Frame duration in milliseconds (default 80ms).
response_format "json" | "binary" How the result should be returned. Defaults to JSON.

Python usage

from app import create_app
from app.services.transformation_service import TransformationRequest, transform
from PIL import Image

app = create_app()
with app.app_context():
    source = Image.open("my_face.png")
    target = Image.open("assets/pfp_transparent.png")
    result = transform(
        TransformationRequest(
            source=source,
            target=target,
            blend_ratio=0.6,
            make_gif=False,
            gif_frame_count=12,
            gif_duration=80,
            max_dimension=1024,
        )
    )
    with open("output.png", "wb") as fh:
        fh.write(result.data)
